# External Masks Evaluator

This standalone tool allows you to compute rigorous segmentation metrics by comparing a directory of ground truth masks against a directory of predicted masks generated by external models or systems.

## Features

- Evaluates **Precision, Recall, F1-Score, and mIoU**.
- Reports **Macro (Mean)**, **Weighted**, and **Micro** averages for a comprehensive overview of performance across classes (including class imbalances).
- Matches files automatically by filename between the directories.
- Ignores pixels outside valid class ranges (e.g., `-1` or `255`).

## Quick Usage

Run the script from the terminal, specifying the paths to your directories and the number of segmentation classes.

```bash
python external_evaluator/evaluate_masks.py \
    --gt_dir "/path/to/ground_truth/masks" \
    --pred_dir "/path/to/predicted/masks" \
    --num_classes 5
```

### Arguments

| Argument        | Description                                                              |
| :-------------- | :----------------------------------------------------------------------- |
| `--gt_dir`      | **(Required)** Path to the directory containing ground truth masks.      |
| `--pred_dir`    | **(Required)** Path to the directory containing the predicted masks.     |
| `--num_classes` | **(Required)** The number of valid segmentation classes in your dataset. |

### Notes

- The tool expects mask images where pixel values represent class indices (e.g., `0`, `1`, `2`, `3`, `4`).
- Supported formats: `.png`, `.tif`, `.tiff`, `.jpg`, `.jpeg`.
- If the tool encounters multi-channel (e.g., RGB) masks, it will attempt to use the first channel.
